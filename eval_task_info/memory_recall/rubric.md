具体指标和说明：
1.Accuracy：是否正确回忆事实（事件、人物、状态、数值等）
    评分标准：
    5分：所有关键信息完全正确，无明显错误
    4分：基本正确，仅有轻微细节错误（不影响理解）
    3分：有部分错误（≈20–40%信息不准确）
    2分：错误较多（≈40–70%不准确）
    1分：大部分信息错误
    0分：完全错误或未回答

2.Completeness：是否覆盖题目要求的关键要点（不是越多越好，而是该有的是否都有）
    评分标准：
    5分：覆盖全部关键点，无遗漏
    4分：仅缺少1个次要信息点
    3分：遗漏部分关键点（约30%）
    2分：遗漏较多关键点（约50%以上）
    1分：只提及极少部分信息
    0分：未涉及核心内容

3.Temporal Consistency：事件顺序是否正确、是否混淆时间关系
    评分标准：
    5分：时间顺序完全正确，无混乱
    4分：基本正确，有轻微顺序偏差
    3分：部分顺序错误，但仍能理解整体脉络
    2分：顺序混乱，影响理解
    1分：严重错乱，几乎无法判断时间关系
    0分：完全忽略时间或全部错误

4.No Hallucination：是否编造不存在的信息（最关键的安全指标）
    评分标准：
    5分：完全没有编造内容
    4分：有极少轻微扩展（不影响事实）
    3分：有少量编造或过度推断
    2分：明显编造内容（但仍有部分真实信息）
    1分：大量虚构或严重夸大
    0分：主要内容为编造
